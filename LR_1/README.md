## Цель работы
Знакомство с инструментами построения пользовательских интерфейсов web-сайтов: HTML, CSS.

В ходе работы был разработан интерфейс калькулятора, стилизованный под портал «Госуслуги», с фиксированной шапкой, возможностью смены темы оформления и навигацией на отдельную страницу с подробным описанием цели.

## Функциональность
- Базовые арифметические операции: сложение, вычитание, умножение, деление.
- Кнопки управления: очистка (C), смена знака (+/-), процент (%).
- Ввод чисел с десятичной точкой.
- Отображение результата на экране калькулятора.
- Фиксированный хэдер с кнопками:
  - «← На главную» (возврат на главную страницу);
  - «Цель ЛР» (переход на отдельную страницу `purpose.html` с полным текстом цели).
- Выбор темы оформления:
  - «Госуслуги (по умолчанию)» – светлая сине-белая гамма.
  - «Тёмная» – тёмный фон, адаптированные цвета.
  - «Светлая» – зелёная светлая тема.
- Сворачивающийся блок «Об авторе» с информацией об исполнителе.
- Футер со ссылкой на репозиторий.

## Технологии
- **HTML5** – семантическая разметка.
- **CSS3** – стилизация, Flexbox, адаптивный дизайн, градиентный фон, анимации наведения.
- **Google Fonts** – шрифт Inter (фирменный шрифт Госуслуг).

## Структура проекта (ЛР1)
lr1/

├── purpose.html        # Страница с целью работы
├── style.css           # Полные стили проекта (включая хэдер и темы)

## Установка и запуск
1. Скачайте все файлы в одну папку.
2. Откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Edge, Safari).
3. Для просмотра страницы цели нажмите кнопку «Цель ЛР» в хэдере.

## Внешний вид
- Калькулятор имеет скруглённые углы, тени, аккуратные кнопки с эффектами наведения.
- Экран калькулятора имитирует ЖК-дисплей.
- Хэдер «приклеен» к верхней части окна и растянут на всю ширину.
- Переключение темы меняет цвета фона, кнопок, границ, а также цвета элементов хэдера.

## Особенности реализации
- Использование CSS-переменных (позже заменено на динамическое применение классов через JS для тем, но в ЛР1 темы реализованы через `id="theme-select"` и переопределение стилей через JavaScript).
- Фиксированное позиционирование хэдера с `padding-top` у `body`, чтобы контент не перекрывался.
- Адаптивная вёрстка: калькулятор имеет максимальную ширину 340 пикселей и центрируется на экране.
